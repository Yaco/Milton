Milton by Yaco

Tool to easily create videos from presentations and conferences where slides were used.

What is it for?

Instructions

  1. Just before starting your presentation, run milton in the lecturer computer:

    $sudo milton "folder name"

    Also start your audio recording, use the tool you preffer or an external device.
  2. You will have 10 seconds until milton starts to work, so go to your presentation software (eg: OpenOffice or a PDF viewer) and start the presentation.
  3. Each time you move forward or backwards in the presentation (using up, down, left, right, avpag, repag keys of your keyboard), milton will capture the screen. Dont use the mouse to control your presentation!
  4. When the presentation is finished, stop the audio recording and stop milton. You will find a folder (with the name indicated in [1]) containing many png files, each per slide change you made. The filenames are sortered according to the time creation of the slide.

  5. Run the script milton2slideshow.sh using "folder name" as the directory argument. 'milton2slideshow.sh' is just a simple fork of dir2slideshow (tool incuded in dvd-slideshow), that introduces the time syncronization. So you can use 'milton2slideshow.sh' in the same way you use 'dir2slideshow', all features and command line options are the same.

    Example: $ milton2slideshow -c 0.5 -n "your presenation title" "folder name"

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/Yaco/Milton

Dependences

  • screenFetch: uses 'scron' to capture make the screenshots
  • dvd-slideshow: used to produce de final video.
